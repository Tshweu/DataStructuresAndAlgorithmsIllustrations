Queues

A queue is a linear

data structure that follows the First In First Out (FIFO) principle.

Elements can be added to the end of the queue and removed from the front.

Queues are used in various applications such as task scheduling, breadth-first search, and managing resources in a system.

Queue Operations

  • Enqueue (Push): Add an element to the end of the queue.
  • Dequeue (Pop): Remove the fist element from the stack.
  • Peek: Retrieve the last element without removing it.
  • isEmpty: Check if the queue is empty.
