Graphs

A graph is a non-linear data structure consisting of nodes (vertices) connected by edges.

Graphs can be directed or undirected, weighted or unweighted, and are used to model various real-world problems.

Applications include social networks, shortest path finding (e.g., Google Maps), recommendation systems, and network topology.

Graph Representations

  • Adjacency Matrix: A 2D array where each cell indicates the presence (and weight) of an edge between vertices.
  • Adjacency List: Each node maintains a list of its adjacent vertices; more space-efficient for sparse graphs.

Common Graph Operations

  • Add Vertex: Insert a new node into the graph.
  • Add Edge: Create a connection between two vertices.
  • Remove Vertex: Delete a node and all its connected edges.
  • Remove Edge: Delete a specific connection between two nodes.
  • Traversal: Visit all nodes in a specific manner (e.g., Depth-First Search, Breadth-First Search).
  • Search: Find a specific node or edge in the graph.
