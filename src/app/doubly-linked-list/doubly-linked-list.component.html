Doubly Linked List

A doubly linked list is a linear data structure in which each element (node) points to both its previous and next node.

This allows traversal in both forward and backward directions.

It is used in applications such as navigation systems, undo functionality in editors, and memory management.

Common Operations

  • Insert: Add a node at the beginning, end, or a specific position.
  • Delete: Remove a node from the beginning, end, or a specific position.
  • Search: Find a node with a specific value.
  • Traversal: Traverse the list forward or backward.
