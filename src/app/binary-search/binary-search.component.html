Binary Search

Binary search is an efficient algorithm used to find an element in a sorted array or list.

It works by repeatedly dividing the search interval in half and comparing the target value to the middle element.

Binary search is significantly faster than linear search for large datasets but requires the input to be sorted.

Binary Search Steps

  • Step 1: Identify the middle element of the sorted array.
  • Step 2: Compare the target value with the middle element.
  • Step 3:
    • If equal, the search is complete.
    • If the target is smaller, repeat the search in the left half.
    • If the target is greater, repeat the search in the right half.
  • Step 4: Repeat until the element is found or the search interval is empty.

Binary Search Properties

  • Time Complexity: O(log n)
  • Space Complexity: O(1) for iterative, O(log n) for recursive
  • Requirement: The input must be sorted
